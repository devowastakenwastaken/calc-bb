=== HELP ===

To perform one of the basic 4 operations, do:
Division = x / y
Addition = x + y
Multiplication = x * y
Subtraction = x - y

Add negative = x + -y
Power = x ** y
Power (second method) = enter 'pow', enter base, enter exponent

To do advanced calculations:
Factorial: enter 'fact', enter target number
Square root: enter 'sqrt', enter target number
Logarithms: enter 'log', enter base, enter number

sin: enter 'sin', enter target angle
cos: enter 'cos', enter target angle
tan: enter 'tan', enter target angle

To get pi: enter 'pi'
To get Euler's number: enter 'e'

Absolute value: enter 'abs', enter target number
Floor and ceiling: enter 'floor' or 'ceil', enter target number

Greatest common denominator / Highest common factor: enter 'gcd', enter first and second number
Lowest common multiple: enter 'lcm', enter first and second number

Dividing a number by 0 will return infinity.
